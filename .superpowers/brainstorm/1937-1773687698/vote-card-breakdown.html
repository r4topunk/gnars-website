Vote card — own vs. delegated breakdown

How should each voter's breakdown appear? Using r4topunk.eth (41 votes: 20 own + 21 delegated) as example.

A — Inline tag after vote count
r4topunk.eth voted FOR with 41 votes 20 own 21 delegated about 2 hours ago

Stays on the same line. Minimal footprint — tags appear right after the vote count. Easy to ignore if you don't care about the breakdown.

B — Secondary line below the header
r4topunk.eth voted FOR with 41 votes about 2 hours ago
20 own tokens · 21 delegated from 3 wallets

Header stays clean. Breakdown lives on a second line indented under the avatar. Delegator count ("from 3 wallets") adds useful context.

C — Mini stacked bar inside the card
r4topunk.eth voted FOR about 2 hours ago
41 votes
■ 20 own ■ 21 delegated

Visual proportion bar shows the ratio at a glance. Best when many voters have delegated power — the split becomes spatially obvious.
