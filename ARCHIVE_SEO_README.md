# Archive Section - SEO Implementation

## Summary

This PR implements an archive section for historical Gnars blog posts to preserve SEO value and prevent 404 errors. The archive posts are **not linked in main navigation** but remain publicly accessible and indexable by search engines.

## Problem

Google Search Console showed **6 blog URLs with 404 errors**:
1. `/sub-dao-culture/`
2. `/best-cc0-nft-projects/`
3. `/history-of-nfts/`
4. `/nfts-music-industry-second-life/`
5. `/on-chain-nfts-and-why-theyre-better/`
6. `/tags/?tag=education`

These posts existed on the old gnarsdotcom site but were removed during the redesign. While they had **zero traffic in the last 12 months**, they contain high-quality content written by @gami_vc and showcase Gnars' thought leadership in the NFT/DAO space.

## Solution

### Architecture Decision: Static Archive (Not Paragraph API)

**Why static?**
- Archive posts are historical (2021-2022) and will never be updated
- No need for CMS overhead
- Faster page loads (no API calls)
- Simpler maintenance
- Clear separation from current blog system

**Current blog system** (`/blogs`) continues using Paragraph API for new content.

### Implementation Details

1. **New Routes**
   - `/archive` - Index page listing all archived posts
   - `/archive/[slug]` - Individual archive post pages

2. **Content Storage**
   - Markdown files in `src/content/archive/`
   - Copied from gnarsdotcom repo with original front matter preserved
   - Rendered server-side with `react-markdown`

3. **SEO Features**
   - ✅ 301 redirects from old URLs → new archive URLs
   - ✅ All pages indexed in sitemap.xml
   - ✅ `<meta name="robots" content="index, nofollow">` on archive posts
     - **index**: Let Google crawl the content
     - **nofollow**: Don't follow links (some may be outdated)
   - ✅ Proper metadata (title, description)
   - ✅ Structured with headings, proper HTML semantics

4. **User Experience**
   - 🔔 Yellow notice banner on all archive pages: "This is historical content, some info may be outdated"
   - 🔗 Link back to main blog
   - 📅 Publication date clearly visible
   - 🏷️ Tag taxonomy preserved from original posts
   - **Not visible in main navigation** (keeps current site clean)

5. **Content Quality**
   All 5 posts are **high-quality, well-researched content**:
   - **Sub-DAO Culture** - Core Gnars positioning, DAO governance
   - **Best CC0 NFT Projects** - Comprehensive overview of CC0 space
   - **History of NFTs** - Evergreen reference material
   - **On-Chain NFTs** - Technical deep-dive
   - **Music + NFTs** - Industry use cases
